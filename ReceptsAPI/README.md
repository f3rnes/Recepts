# Книга рецептов

## Функционал
* Регистрация, логин
* Разделение пользователей по ролям: юзер и админ
* Админ может добавлять, редактировать, удалять рецепты, удалять комментарии
* Пользователь может просматривать рецепты и добавлять, удалять комментарии

## Таблицы
* пользователи: id, имя, фамилия, аватарка, роль, логин (ник), пароль
* рецепты: id, название, описание, фотография, вес блюда, ингридиенты, id админа
* стадии готовности: id, id рецепта, номер стадии, фото, описание
	* комментарий: id, id рецепта, id пользователя, позитивный или нет (бул), текст комментария
